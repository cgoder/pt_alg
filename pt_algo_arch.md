# 训练计划算法-架构

------

## 第一层：用户目标层

输入参数：

- 主目标（增肌/减脂/力量/健康）
- 训练频率
- 可用时长
- 器械条件
- 恢复能力等级（可后续评估产品接入）

输出：

> 目标权重向量

------

## 第二层：刺激模型选择层

根据目标权重选择模型结构：

| 模型 | 主变量   |
| ---- | -------- |
| 力量 | 强度优先 |
| 增肌 | 容量优先 |
| 代谢 | 密度优先 |
| 控制 | 技术优先 |

输出：

> 变量权重矩阵

------

## 第三层：负荷调控层

建立边界：

- MEV（最小有效量）
- MAV（最大适应量）
- MRV（最大恢复量）

控制变量：

- Volume
- Intensity
- Density
- Frequency

输出：

> 安全区间内的训练负荷曲线

------

## 第四层：周期波动层

引入：

- 微周期（周）
- 中周期（4-6周）
- 卸载机制
- 强弱交替

输出：

> 时间波动模型

------

## 第五层：单课生成层

- 动作选择
- 组数
- 次数
- 节奏
- 轮次
- 休息时间
